## 🚀 Quick Start

### 1. Clone Repository
```bash
git clone https://github.com/yourusername/blais-beats-store.git
cd blais-beats-store
```

### 2. BeatStars Setup (5 minutes)
1. Create BeatStars account at [beatstars.com](https://beatstars.com)
2. Upload your beats to BeatStars
3. Create Blaze Players for each beat and copy embed codes
4. Add beats to your store using the admin panel

### 3. Launch Store
- Open `index.html` in browser for customer store
- Open `admin-simple.html` for beat management
- Deploy to GitHub + Vercel for live site

**No backend setup required!**

## 🛠️ BeatStars Setup Guide

### Step-by-Step Integration

1. **Create BeatStars Account**
   - Sign up at [beatstars.com](https://beatstars.com)
   - Complete your producer profile

2. **Upload Your Beats**
   - Upload WAV, MP3, and stems to BeatStars
   - Set your pricing and licensing options
   - Add artwork and descriptions

3. **Create Blaze Players**
   - Go to Features > Players in BeatStars Studio
   - Create a new Blaze Player for each beat
   - Customize player appearance
   - Copy the embed code

4. **Add to Your Store**
   - Open `admin-simple.html`
   - Fill in beat metadata (title, BPM, genre, price)
   - Paste BeatStars embed code in the designated field
   - Save beat - it will appear in your store immediately

5. **Deploy Your Store**
   - Push to GitHub
   - Connect to Vercel for automatic deployment
   - Your store is live with professional BeatStars integration!
